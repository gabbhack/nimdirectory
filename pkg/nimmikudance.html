version control

First digit for PMX
Second digit for PMD
Third for VMD\

0 = non-complete/ no support for such file extension
1-5 = bug fixes
6 = full support for the file extension
6+ = bug fixes\

So 0.0.0 = non-complete/no support for all extensions listed above.
